Sobre


Este projeto pessoal elaborado para prática dos conhecimentos adquiridos de desenvolvimento Front-End, e utiliza as seguintes tecnologias:
  • Angular, versão 12.2.0;
  • Ngx-Bootstrap versão 7.0.0, com Bootstrap versão 4.5.3;
  • Ngx-Mask, versão 12.0.0.
Este projeto também estará sujeito a refatorações, conforme novos conhecimentos forem adquiridos no futuro.
Versão do Projeto: 1.0.1
Última atualização: 09 de Dezembro de 2021.
